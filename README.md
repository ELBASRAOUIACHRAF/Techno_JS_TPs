#  Technologie JavaScript – Travaux Pratiques

##  Description

Ce repository regroupe l’ensemble des Travaux Pratiques (TP) réalisés dans le cadre du module **Technologie JavaScript**.  
Il contient des exercices et des mini-projets visant à maîtriser les concepts fondamentaux et avancés de JavaScript.

L’objectif est de :

- Comprendre les bases du langage (variables, types, fonctions, boucles, conditions)
- Manipuler le DOM
- Gérer les événements
- Utiliser les fonctionnalités modernes 
- Travailler avec la programmation asynchrone (Promises, Async/Await)

---

##  Technologies utilisées

- JavaScript 
- HTML
- CSS
